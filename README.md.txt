# Historical Data Breaches

## Overview
A Python command-line application that retrieves historical data breach information using the Have I Been Pwned API.

## Features
- Search historical data breaches
- Fetch live data from the API
- Clean HTML tags using Regular Expressions (Regex)
- Display breach details in the terminal

## Technologies Used
- Python
- Requests
- Regex
- REST API

## How to Run

```bash
pip install requests
python main.py
```

## Note
This repository is maintained for learning purposes to understand API integration and Regular Expressions in Python.
